# Java Functional Learning Path

Welcome to my journey toward mastering functional programming in Java. 

I am currently focusing on **Java Streams** as my primary gateway into functional paradigms. My goal is to move from basic procedural loops to expressive, declarative, and efficient data processing.

## Current Focus: Java Stream API
I am following a 2-day intensive "0 to Hero" roadmap that covers:
- Core Operations (`map`, `filter`, `sorted`, `distinct`)
- Terminal Operations (`reduce`, `collect`, `find`, `match`)
- Advanced Concepts (`flatMap`, `Optional`, custom collectors)
- Domain-driven data processing (Real-world scenarios with `Employees`, `Orders`, etc.)

## Learning Philosophy
I follow a strict discipline for every problem I solve:
1. **Stream-First:** Solve using Java Stream API.
2. **Analysis:** Identify input, expected output, and operations used.
3. **Critique:** Always compare the Stream approach with an alternative (usually a loop) to evaluate readability and performance.
4. **Maintenance:** Focus on data flow rather than syntax.

## Project Structure
- `src/streamapi/Input.java`: Contains the domain data and record definitions used for all exercises.
- `src/streamapi/Exercises.java`: My workspace for solving the roadmap problems.
- `src/streamapi/todo.txt`: My progress tracker and roadmap.
