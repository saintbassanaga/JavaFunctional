JAVA STREAM - 2 DAY TODO PLAN
From 0 to Hero Practice Roadmap

Goal:
Master Java Stream fundamentals, transformations, filtering, mapping, sorting,
collectors, grouping, aggregation, Optional handling, and real-world data processing.

Rules for your practice:
1. Solve every problem using Java Stream first.
2. After solving, explain in one sentence what each pipeline stage does.
3. For every problem, write down:
   - input
   - expected output
   - stream operations used
   - one alternative approach
4. If a problem feels hard, first solve it with loops, then rewrite with Stream.
5. Do not memorize syntax blindly. Focus on data flow.
